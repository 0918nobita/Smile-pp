Arrayモジュール

  • implode$ 配列の要素を結合して文字列にする
  • indexOf / indexOf$ 配列の要素を検索する

以下、関数名の接頭辞が arr- のものは『疑似多次元配列』を扱う関数となっています。

  • arrDump 疑似多次元配列の情報を出力する
  • arrFlatten 疑似多次元配列の次元数を1次元にする
  • arrSize 疑似多次元配列の要素数を取得する
  • arrUniq 疑似多次元配列の重複要素を排除する

implode$ - 配列の要素を結合して文字列にする (Version 2017.1 ～)

string implode$ ( array array$, string glue$ )

array　連結したい文字列の配列
glue$　各要素間に挟まれる文字列

トランスパイルを行う場合、第二引数 glue$ は省略可能です。(デフォルト値は空文字)

返り値　すべての配列要素の順序を変えずに、各要素間に glue$ をはさんで 1 つの文字列にして返します。

定義コード　厳格モードで実行する場合の定義コードです

def implode$(array$[], glue$)
	var i
	var rv$
	for i = 0 to len(array$) - 1
		if i == 0 then
			rv$ = array$[i]
		else
			rv$ = rv$ + glue$ + array$[i]
		endif
	next
	return rv$
end
